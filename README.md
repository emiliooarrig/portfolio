# Portafolio Personal - Emilio Guzmán

Este repositorio contiene el código fuente de mi portafolio personal, diseñado para destacar mi perfil como estudiante de Ingeniería en Sistemas.

El proyecto es una aplicación web de una sola página (SPA) construida con tecnologías modernas, priorizando el rendimiento, la accesibilidad y un diseño visual "Dark Mode" elegante.

## Tecnologías Utilizadas

- **React**: Biblioteca principal para la construcción de la interfaz de usuario.
- **Vite**: Entorno de desarrollo y empaquetador para una experiencia rápida y optimizada.
- **CSS3 (Vanilla)**: Estilos personalizados sin dependencia de frameworks pesados, utilizando variables CSS para la gestión de temas.
- **Intersection Observer API**: Para las animaciones de "scroll-reveal" sin librerías externas.

## Características Principales

*   **Diseño Responsivo**: Adaptable a dispositivos móviles y de escritorio.
*   **Tema Oscuro (Dark Mode)**: Paleta de colores "Deep Navy" profesional con acentos brillantes para facilitar la lectura.
*   **Animaciones Suaves**: Efectos de desvanecimiento (fade-in) y deslizamiento al hacer scroll.
*   **Navegación Fluida**: Menú de navegación fijo con desplazamiento suave a las secciones.
*   **Secciones Dedicadas**:
    *   **Inicio**: Presentación breve y CTA.
    *   **Sobre Mí**: Background y enfoque profesional.
    *   **Habilidades**: Visualización moderna mediante "tags" de mis competencias en Datos, Desarrollo y DevOps.
    *   **Proyectos**: Cards interactivas con mis trabajos más relevantes.
    *   **Contacto**: Información para conectar.

## 📂 Estructura del Proyecto

```
/src
  ├── assets/        # Recursos estáticos (imágenes, iconos)
  ├── App.jsx        # Componente principal y estructura de la página
  ├── index.css      # Estilos globales y variables de tema
  └── main.jsx       # Punto de entrada de la aplicación React
```
